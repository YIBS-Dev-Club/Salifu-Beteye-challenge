    INSTALLATION OF DAVINCCI RESOLVE AND CHALLENGES
Installation Steps
1.	Open a web browser and go to the Black magic Design official website.
2.	Download DaVinci Resolve for Windows.
3.	Extract the downloaded ZIP file.
4.	Double-click the installer (.exe) file to start installation.
5.	Accept the license agreement and follow the setup instructions.
6.	Click Install, wait for completion, and then launch DaVinci Resolve.
Challenges
• Large download size may take time on slow internet.
• Requires a strong GPU and sufficient RAM for smooth performance.
• Installation issues may occur if Windows or graphics drivers are out-dated
